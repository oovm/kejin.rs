## 简单强化模型

从 + 0 到 + 10;

各个等级独立且可加

$$\sum_{i=1}^{n}{\frac{1}{p_i}}$$

## 简单强化消耗模型

接下来考虑强化的消耗, 假设每强化一级需要消耗 $(\mathrm{level} + 1)^2 × 100$ 的金币, 那么强化到 `+10` 需要的金币期望是多少?.

直接用金币消耗乘每一级需要的期望即可.

$$\sum_{i=1}^{n}{\frac{c_i}{p_i}}$$

// {100, 544, 1669, 3955, 8122, 15322, 27572, 48905, 89405, 189405}
